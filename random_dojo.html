Why, and for Whom?

One of my Krav Maga instructors needed it. I make no claims that anyone else will find it useful or correct. Use at your own risk!

How does the Cooldown feature work?

The page always remembers the name of every skill it has selected since the page was last reloaded. If you have selected to disallow repeats for x turns, skills chosen up to x turns ago will be locked out. Changing this setting does not affect memory. If you turn it on and disallow the last x turns' worth of skills, and those skills themselves include some repeats, fewer than x skills will be locked out at first.
